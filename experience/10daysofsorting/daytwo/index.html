Day 2 of Sorting

Valley Sort

Valley sort keeps sweeping from left to right, looking for a drop where one number is larger than the number beside it. When it finds that little valley, it swaps the pair and starts scanning from the beginning again until the whole list rises smoothly from left to right.

Python Code

# valley sort
# find a drop between neighboring elements
# swap the pair
# restart the scan until no more drops are found

def valleysort(lst: list) -> list:
    sel = 0
    while sel < len(lst) - 1:
        if lst[sel] > lst[sel + 1]:
            lst[sel], lst[sel + 1] = lst[sel + 1], lst[sel]
            sel = 0
        else:
            sel += 1
    return lst
