Inflatiq

Inflatiq is an inflation calculator project I started about a year ago. This page is here to share a little bit about the project and provide a place to download it.

The goal of Inflatiq was to make it easy to compare the value of money across time and better understand how inflation changes purchasing power.

Project Status

Inflatiq is not currently under active development, but it remains a project I learned a lot from and may revisit in the future.
